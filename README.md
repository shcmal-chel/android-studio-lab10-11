Научиться применять принципы Material Design в Android приложениях с использованием Jetpack Compose. В результате лабораторной работы  будет создано приложение «Список студентов», которое демонстрирует использование  цветовых схем, типографики, форм и анимаций Material Design.
  
___
## Теоретическая часть

**Material Design** — это система дизайна, разработанная Google для создания 
качественных цифровых продуктов для Android, веб и других платформ. Она 
предоставляет рекомендации по созданию читабельного, привлекательного и 
согласованного пользовательского интерфейса.

## Основные компоненты Material Theming

* Primary — основной цвет приложения (кнопки, основные элементы)
* Secondary — второстепенный цвет (акценты, FAB) 
* Tertiary — третичный цвет (дополнительные акценты)
* Surface — цвет поверхностей (карточки, листы) 
* Background — фоновый цвет 
* Error — цвет для ошибок

## Примеры кода

```kotlin
 @Composable
fun StudentApp() {
    Scaffold(
        topBar = {
            StudentTopAppBar()
        }
    ) { it ->
        LazyColumn (contentPadding = it) {
            items(students) {
                StudentItem(
                    student = it,
                    modifier = Modifier
                        .padding(dimensionResource(R.dimen.padding_small))
                )
            }
        }
    }
}
```
